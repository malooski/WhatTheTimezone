# What The Timezone

> Coordinate with your friends in other timezones using this simple tool!

### Disclaimer

This tool is in what I'd consider a proper beta. Timezones are hard and there could be some issues/edge-cases not covered yet.

### Features
- Add multiple friends to manage.
- Convert either the current time or a provided time to your friends' timezones.
- Convert a friend's time to your other friends' and your own timezone.
